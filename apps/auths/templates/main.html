Welcome to my web service

Тема проекта: Магазин скинов для DOTA 2

Цель проекта: защита диплома и возможно запуск приложения в продакшн для заработка, но это не точно.

Проект построен на REST архитектуре, запущен на сервисе Google Cloud Platform.
При написании проекта были использованы технологии:
  • Django
  • Django Rest Framework
  • SimpleJWT
  • Docker
  • Docker-Compose
  • Nginx
  • PostgreSQL
  • Celery
  • Redis

Проблемы с которыми я столкнулся:
  • Система оплаты - Внезапно, чтобы оно работало нужно использовать протокол HTTPS для которого требуется:
    • ssl-сертификат
    • доменное имя - которое гугл мне не дает поскольку квота достигнута хотя я ни разу не использовал его

Краткое описание возможностей:
  • можно покупать скины, в общем-то для этого сие мероприятие и делалось
  • можно оставлять отзывы на скины, при покупке надо обязательно поставить оценку от 1 до 5 баллов, рейтинг скина будет пересчитываться с каждым новым отзывом
  • покупка происходит только с виртуального кошелька пополнить который можно было бы если бы я получил домен от гугла...
  • так же можно добавлять других пользователей в список друзей
